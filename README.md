## Getting Started

Install package

```bash
npm i
# or
yarn
# or
pnpm i
```

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

and config env:

```bash
NEXT_PUBLIC_API={API ENDPOINT}
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
